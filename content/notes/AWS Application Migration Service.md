AWS Application Migration Service(MGN)は、オンプレミスや他クラウドで動作しているサーバー全体をAWSにリフトアンドシフト方式で迅速かつ効率的に移行するサービスである。

リフトアンドシフトとは、システムの構成やコードを大きく改変することなく、現状のまま新しい環境へ移行する手法を指す。
