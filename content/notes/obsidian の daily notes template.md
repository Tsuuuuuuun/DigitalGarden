## create notes

```dataview
table file.link as "ノートタイトル",
      date(file.cday) as "作成日時"
from "notes"
where date(file.cday) = date(this.file.name)
sort file.ctime asc
```

## modified notes

```dataview
table file.link as "ノートタイトル",
      date(file.cday) as "作成日時",
      date(file.mday) as "編集日時"
from "notes"
where date(file.mday) = date(this.file.name) and date(file.cday) != date(this.file.name)
sort file.mday desc
```
